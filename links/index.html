Resources Used

For the patterned backgrounds on this site I have used heropatterns.com by Steve Schoger, a very cool resource for customizable, SVG-based repeating patterns. They are provided under a Creative Commons attribution license.

The font used in my site banner is Jacquard 24 by Sarah Cadigan-Fried. It is provided under the Open Font License and is open-source. Sarah has made several very cool pixel-type fonts, which are all available on her GitHub profile linked below:

  • Sarah Cadigan-Fried’s GitHub profile
  • License for Jacquard
  • The repository for Jacquard on GitHub

This site is built using Hugo, a simple and lightweight static-site generator (SSG) written in Go. You can learn more about Hugo at gohugo.io. My Hugo theme is adapted from lugo by Luke Smith. You need a theme to get started with Hugo, and lugo is a very minimal sort of tutorial theme that has enough to get you going while still being easy to rewrite to meet your needs. See Luke’s Hugo tutorial on YouTube.
